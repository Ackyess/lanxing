# 揽星 · HR 招聘 AI 助手

面向 HR 的 Boss 直聘辅助 Chrome 扩展：用你自己的 AI API（OpenAI 兼容格式）对牛人列表做简历初筛，通过的人才自动进入人才池，支持横评排序和导出。

- 无构建、无框架的 MV3 扩展，克隆即可加载
- AI 走你自己配置的 API（Token 只存在本地浏览器，不经过任何第三方服务器）
- 三种使用形态：工具栏弹窗 / 浏览器侧边栏 / 页面内可拖动浮窗

## 安装

1. 下载或克隆本仓库
2. 打开 `chrome://extensions/`，右上角开启「开发者模式」
3. 点「加载已解压的扩展程序」，选择仓库根目录
4. 工具栏出现「揽星」图标即安装完成

## 快速上手（4 步）

1. **配置 AI**：点扩展图标 → 右上角「设置」→ 填入你的 API Token（可选填第三方 API 地址，OpenAI 兼容格式）。填完 Token 后模型下拉框会**自动从上游拉取可用模型列表**，选一个视觉模型（需要能看图，如 Qwen3-VL）；上游不支持时会回退到预设列表，也可以选「自定义模型...」手动填。
2. **同步岗位**：打开 Boss 直聘并登录 → 回到面板点「一键同步岗位」，你的在招岗位会进入岗位库；也可以在「管理面板」手动增删岗位和 JD。
3. **设置队列**：在「运行队列」里选岗位、填数量（1-200），可以点 + 排多个岗位依次跑。
4. **开始运行**：打开 Boss 直聘的**牛人推荐列表页**，点「AI 辅助分析简历」。不确定链路是否通时先点「单人试跑」验证一个候选人。

AI 判定通过的候选人会自动进入「通过人才池」，随后可「AI 横评排序」（按当前岗位横向对比给出优先级）或「导出通过人才」（JSON）。

## 三种使用形态

| 形态 | 打开方式 | 适合 |
| --- | --- | --- |
| 弹窗（默认） | 点工具栏图标 | 快速查看，800×600 双栏 |
| 侧边栏 | 设置 → 面板显示方式 → 浏览器侧边栏 | 与牛人列表并排、点页面不关闭、宽度可拖 |
| 页内浮窗 | 面板顶部「浮窗」按钮（需在 Boss 直聘页面上） | 跑批时拖到角落盯日志，位置有记忆 |

三种形态共用同一份配置、人才池和日志，随时互切。

## 常见问题

- **连接失败 HTTP 401**：Token 无效或没有权限，检查设置里的 Token 与 API 地址是否匹配。
- **模型下拉没有自动更新**：上游需要支持 `GET /v1/models`（OpenAI 兼容端点）；不支持时用预设或自定义模型即可，运行日志里会有提示。
- **浮窗按钮点了没反应**：浮窗只能注入 Boss 直聘域名的页面；刚更新扩展后需要刷新一次 Boss 直聘页面。
- **侧边栏切不回弹窗**：在侧边栏里打开设置，把「面板显示方式」改回「工具栏弹窗」。
- **运行日志在哪**：面板右栏（弹窗）或底部（侧边栏），保留最近 100 条，重开面板还在。

## 开发

- 目录即源码，直接加载，无 bundler；改完在 `chrome://extensions/` 点扩展卡片上的 ⟳ 重载（改了 content script 还需刷新 Boss 直聘页面）
- `npm run check`：提交前跑 validate + test + syntax
- `npm run package`：打包 zip 到 `dist/`（已 gitignore）

```
manifest.json        MV3 manifest
background.js        service worker：帧路由、截图、面板显示方式
content_scripts/     Boss 直聘页自动化、数据提取、页内浮窗
popup/               面板 UI（弹窗/侧边栏/浮窗共用）+ 模块脚本
utils/               AI 请求、图像、提示词、鉴权等工具
docs/                内部计划与恢复记录
```
